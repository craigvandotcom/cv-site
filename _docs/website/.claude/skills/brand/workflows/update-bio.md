# Update Platform Bio Workflow

**Purpose:** Step-by-step process for updating bios across platforms
**Owner:** Brand skill + content-creator agent
**Frequency:** Quarterly review, ad-hoc updates
**Last Updated:** 2025-12-31

---

## When to Update Bios

### Triggers for Bio Updates

**Mandatory updates:**
- New product launch (add to CTA)
- Major positioning shift (new tagline, audience)
- Subscriber milestone (10K → 50K → 100K)
- Brand evolution (integrity framework expansion)

**Quarterly review triggers:**
- Platform character limits changed
- Audience response data (A/B test winners)
- Competitive positioning shift
- New platform added to strategy

**Ad-hoc updates:**
- User asks "update my bio"
- Preparing for podcast appearance (guest bio needed)
- Platform-specific campaign launch
- Voice/tone refinement from testing

---

## Pre-Update Checklist

Before starting workflow:

- [ ] Identify which platform(s) need updating
- [ ] Review current bio performance (if available)
- [ ] Check latest brand positioning strategy
- [ ] Verify character limits haven't changed
- [ ] Determine audience focus (optimizer vs seeker)
- [ ] Confirm current CTA priority (newsletter/book/app)

## Step-by-Step Workflow

### Step 1: Load Templates and Context

**Files to read:**
- `.claude/skills/brand/templates/platform-bios.md` (bio templates)
- `.claude/skills/brand/SKILL.md` (core brand identity)
- `.claude/skills/brand/reference/positioning-strategy.md` (audience context)
- `.claude/memory/brand.md` (recent brand evolution notes)

**Extract:**
- Current platform bio (if exists)
- Character/word limits
- Platform-specific template
- Positioning angle (optimizer/seeker/universal)

---

### Step 2: Select Positioning Angle

**Questions to answer:**

1. **Which audience is primary for this platform?**
   - LinkedIn → Optimizers (professional)
   - Instagram → Seekers (spiritual/visual)
   - X/Twitter → Universal (systems thinking)
   - YouTube → Universal (broad reach)
   - Newsletter → Universal (dual audience)

2. **What's the conversion goal?**
   - Newsletter signup (most common)
   - Book download (permafree funnel)
   - App download (product trial)
   - Speaking/consulting inquiry (professional)

3. **What voice matches platform norms?**
   - LinkedIn: Professional authority
   - Instagram: Casual vulnerability
   - X: Tactical/contrarian
   - YouTube: Educational/conversational
   - Newsletter: Systems thinking + story

---

### Step 3: Draft New Bio

**Using template as starting point:**

1. Copy relevant template from `platform-bios.md`
2. Customize for current context:
   - Update metrics (subscriber count, user count)
   - Adjust positioning angle if needed
   - Ensure tagline included ("health systems: eat, move, meditate")
   - Include transformation language (Puppet → Pilot)
   - Add specific CTA with link

3. Check against brand requirements:
   - Sovereignty messaging (not guru positioning)
   - Systems focus (Eat, Move, Meditate)
   - Integrity alignment (if appropriate for platform)
   - Anti-dogma tone (where relevant)

---

### Step 4: Apply Writing Guidelines

**Voice check (from writing-guidelines skill):**

- [ ] Passes coffee test (would you say this to a friend?)
- [ ] No em dashes (use periods or commas)
- [ ] Active voice throughout
- [ ] Health jargon translated to human language
- [ ] 1-2 sentences per paragraph (mobile-first)
- [ ] Reads aloud without gasping

**Platform-specific adjustments:**
- LinkedIn: More formal, professional credentials
- Instagram: More emojis (if using), visual language
- X: Punchier, contrarian edge
- YouTube: Conversational, benefits-focused
- Newsletter: Systems thinking, depth

---

### Step 5: Verify Character/Word Limits

**Character count tools:**
- Use character counter for short bios (Twitter, Instagram)
- Word count for longer bios (Newsletter, Podcast)
- Leave buffer (don't max out - updates need room)

**Platform limits:**
| Platform | Limit | Target Range |
|----------|-------|--------------|
| Twitter/X | 160 chars | 140-155 |
| LinkedIn | 2600 chars | 2000-2500 |
| Instagram | 150 chars | 130-145 |
| YouTube | 5000 chars | 3500-4500 |
| Newsletter | ~50 words | 40-50 |
| Threads | ~500 chars | 400-480 |

**If over limit:**
- Remove unnecessary modifiers
- Condense transformation language
- Shorten CTA phrasing
- Consider universal variant instead of audience-specific

---

### Step 6: Test Against Brand Filters

**Strategic filters (from positioning-strategy.md):**

1. **Sovereignty Test:** Does this empower or create dependency?
   - ✅ "I teach you to read signals"
   - ❌ "Follow my protocol"

2. **Transformation Test:** Does this show Puppet → Pilot shift?
   - ✅ "From Patient to Operator"
   - ❌ Generic "get healthier"

3. **Systems Test:** Does this reference frameworks, not just outcomes?
   - ✅ "Three systems: Eat, Move, Meditate"
   - ❌ "Lose weight, feel great"

4. **Audience Test:** Is this for optimizers, seekers, or universal?
   - Verify language matches intended audience

5. **Anti-Dogma Test:** Does this avoid guru positioning?
   - ✅ "Guide back to your own authority"
   - ❌ "Trust my expertise"

**If any test fails, revise bio.**

---

### Step 7: Compare Against Current Bio

**If updating existing bio:**

1. **What changed?**
   - New product/offering
   - Positioning refinement
   - Audience focus shift
   - Metrics update
   - Voice evolution

2. **What stays the same?**
   - Core tagline (always keep)
   - Transformation language (always include)
   - Brand identity (integrity focus)

3. **Is this an improvement?**
   - Clearer value proposition?
   - Stronger CTA?
   - Better audience targeting?
   - More authentic voice?

**If not clearly better, don't change.**

---

### Step 8: Deploy and Document

**Deployment:**
1. Copy final bio to clipboard
2. Log into platform
3. Paste and format (check line breaks, links)
4. Preview before saving
5. Save/publish

**Documentation:**
1. Update `.claude/memory/brand.md` with change note:
   ```
   - [2025-12-31] Updated X bio: Added integrity focus, shifted to optimizer positioning
   ```

2. Update `templates/platform-bios.md` if this becomes new template:
   - Replace old template with new version
   - Note what changed and why
   - Include character count

3. If major positioning shift, note in `reference/positioning-strategy.md`:
   - What changed
   - Why (data, insight, evolution)
   - Impact on other platforms

## Multi-Platform Update Strategy

**When updating multiple platforms:**

### Batch Update Approach
1. Start with universal positioning (YouTube, Newsletter)
2. Create optimizer variant (LinkedIn, X)
3. Create seeker variant (Instagram)
4. Ensure consistent core across all (tagline, transformation)
5. Deploy in order of importance

### Consistency Checks
- [ ] Tagline identical across all platforms
- [ ] Transformation language consistent (Puppet → Pilot)
- [ ] CTAs lead to same primary funnel (usually newsletter)
- [ ] Metrics consistent (subscriber count, user count)
- [ ] Voice adapts to platform but brand stays recognizable

## A/B Testing Bios

**For platforms allowing bio variants (ads, campaigns):**

### Test Variables
- Optimizer vs Seeker positioning
- Systems-focused vs Philosophy-focused
- Benefit-driven vs Identity-driven
- With/without specific metrics

### Test Structure
1. Create 2 variants (only 1 variable different)
2. Run for 30-90 days
3. Track conversion to newsletter/book/app
4. Winner becomes new template

### Example Test
**Variant A (Optimizer):**
"Teaching mission-driven men to eliminate symptoms through body literacy. health systems: eat, move, meditate [link]"

**Variant B (Seeker):**
"Food as meditation. Body as teacher. Personal sovereignty through signal awareness. health systems: eat, move, meditate [link]"

**Measure:** Newsletter signup rate from bio click

## Common Mistakes and Fixes

| Mistake | Fix |
|---------|-----|
| Too generic ("health coach") | Lead with specific transformation (Puppet → Pilot) |
| Missing tagline | Always include "health systems: eat, move, meditate" |
| Guru tone ("I'll fix you") | Sovereignty tone ("I'll teach you to read signals") |
| Wrong audience for platform | Match optimizer/seeker to platform demographics |
| No CTA or vague CTA | Specific benefit-driven CTA ("Weekly systems thinking") |
| Over character limit | Remove modifiers, condense, use abbreviations wisely |
| Jargon-heavy | Translate to human language (eliminate → remove) |
| Feature-focused | Benefit and transformation-focused |

## Platform-Specific Pro Tips

### Twitter/X
- Use line breaks for scanability (even in short bio)
- Consider emoji for visual break (but not required)
- Link should be newsletter (highest ROI)

### LinkedIn
- Use all available space (2600 chars is a lot)
- Include professional credentials/track record
- Multiple CTAs okay (newsletter + book + consulting)
- Professional but not corporate tone

### Instagram
- Emoji usage more accepted (optional)
- Visual language ("See the signals")
- Simple CTA ("Link in bio" standard)

### YouTube
- Front-load value proposition (first 200 chars)
- Explain channel structure (SYSTEMS vs PHILOSOPHICAL)
- Include disclaimer if needed (not medical advice)
- Long enough to be searchable (keywords matter)

### Newsletter
- Hook reader immediately (first sentence critical)
- Promise + personality balance
- Set expectation for frequency/style

## Emergency Bio Update

**When you need bio FAST (podcast appearance, campaign launch):**

1. **Use template verbatim** (customize metrics only)
2. **Skip A/B testing** (use proven universal variant)
3. **Verify character limit** (don't skip this)
4. **Deploy immediately**
5. **Document later** (but do document)

**Quick template selection:**
- Professional context → LinkedIn template
- Casual context → Instagram template
- Podcast → Podcast guest bio (100 words)
- Unknown audience → Universal variant (Twitter/YouTube)

## Quarterly Bio Review Process

**Every 90 days:**

1. **Audit current bios across all platforms**
   - Are they consistent?
   - Do they reflect current positioning?
   - Are metrics up to date?
   - Are links working?

2. **Review brand memory for evolution**
   - Read `.claude/memory/brand.md`
   - Check `reference/positioning-strategy.md`
   - Note any strategic shifts

3. **Check performance data (if available)**
   - Which bios drive most conversions?
   - Which platforms growing fastest?
   - Any A/B test results?

4. **Make batch updates if needed**
   - Use multi-platform workflow
   - Document all changes
   - Update templates

5. **Plan next quarter tests**
   - What to A/B test?
   - New platform to add?
   - Positioning angle to try?

## Success Metrics

**How to know bio is working:**

1. **Conversion rate:** Bio views → newsletter signups
2. **Clarity test:** Can stranger explain what you do after reading?
3. **Recognition test:** Does bio feel like brand voice?
4. **Engagement:** Are followers aligned with target audience?
5. **Professional inquiries:** Are partnership/speaking requests aligned?

**Red flags:**
- High views, low conversions (unclear CTA or value prop)
- Wrong audience engaging (positioning mismatch)
- Frequent confusion questions ("what do you actually do?")
- Feels generic or forgettable
