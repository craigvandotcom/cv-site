## Usage Notes

### When to Use Optimizer Variant
- LinkedIn (professional context)
- Podcast intros (performance audience)
- X threads on systems thinking
- Content targeting founders/executives

### When to Use Seeker Variant
- Instagram (visual/spiritual content)
- Substack philosophical essays
- Spiritual podcast appearances
- Content targeting conscious living audience

### When to Use Universal/Hybrid
- Newsletter (serves both audiences)
- YouTube (broad reach)
- Book descriptions (wide appeal)
- General brand positioning

### Customization Guidelines

1. **Always include:** "health systems: eat, move, meditate" (core tagline)
2. **Always show transformation:** Puppet → Pilot (or variant)
3. **Platform voice:** Adjust formality to platform norms
4. **CTA specificity:** Link to most relevant next step
5. **Character limits:** Use tool to verify before posting
6. **Update frequency:** Review quarterly, update as brand evolves
