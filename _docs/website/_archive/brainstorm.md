## The Full neoMeta Architecture

### The Three Meta-Domains

| Domain | Purpose | Question |
|--------|---------|----------|
| **Introduction** | WHY | Why does neoMeta exist? Why now? Why this approach? |
| **Core** | WHAT | What are we optimizing for? |
| **Foundations** | HOW | How do we implement it? |

### Introduction Domain (Philosophy - No Pillars)

The philosophical foundation:
- **Time Preference Thesis** - The pattern beneath all good choices
- **AI Forcing Function** - Why this matters now
- **Decentralized Alignment** - Who defines your highest good
- **Integrity as OS** - What makes the system function
- **Internal Revolution** - Where transformation happens

### Core Domain (9 Pillars)

| Domain | Archetype | Pillars | Model |
|--------|-----------|---------|-------|
| **Health** | Warrior | Eat, Move, Meditate | Input → Output → State |
| **Wealth** | Magician | Mastery, Mission, Money | Learn → Contribute → Receive |
| **Connection** | Lover | Play, Community, Intimacy | Joy → Belonging → Depth |

### Foundations Domain (3 Pillars)

| Pillar | Archetype | Question |
|--------|-----------|----------|
| **Cycles** | King | WHEN to focus on what |
| **Systems** | King | HOW to execute |
| **Support** | King | WHO/WHAT helps you |

### The Compound of Compounds

- Each pillar amplifies the others within its domain
- Each domain amplifies the other domains
- Low time preference choices compound over time AND over each other
- This creates exponential transformation

## Design Direction

### Visual Philosophy
- Dark mode aesthetic (matches developer + premium health audience)
- Card-based layout with rounded corners
- Gradient accents with green glow effects
- Glass-morphism effects
- Subtle animations (orbiting elements, hover states)
- Blueprint aesthetic with fine linework

### Color Palette

**Core Colors:**
| Name | Hex | Usage |
|------|-----|-------|
| **Deep Charcoal** | `#1a1a1a` | Primary background |
| **Warm Cream** | `#F5F0E6` | Text, light elements |
| **Brand Green** | `#01A878` | Accent, glow effects, emphasis |
| **Deep Charcoal Green** | `#0D3D3D` | Background undertones |

**Secondary:**
| Name | Hex | Usage |
|------|-----|-------|
| **Bitcoin Orange** | `#F7931A` | Wealth/Money topics only |

**Zone Colors (Body Compass):**
| Zone | Hex |
|------|-----|
| Red Zone | `#D62828` |
| Yellow Zone | `#FCBF49` |
| Green Zone | `#2D6A4F` |

### Typography
- Headlines: Bold condensed sans-serif (Space Grotesk)
- Body: Clean readable sans-serif (Inter)
- Code/System elements: Monospace
- High legibility, clear hierarchy
